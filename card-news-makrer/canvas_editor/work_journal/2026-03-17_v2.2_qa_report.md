## 발견된 버그

### BUG-001: Step 전환 시 콘텐츠가 보이지 않음 (CRITICAL → FIXED)

**증상:** Step 1에서 주제 선택 후 Step 2로 전환 시, 콘텐츠가 투명(opacity: 0)한 채 렌더링됨. DOM에는 존재하지만 시각적으로 보이지 않음.

**원인:** `CreateWizard.tsx`의 `StepContent` 컴포넌트에서 fade 전환 로직 문제.
- `setDisplayStep(step)` 호출 → React re-render → effect cleanup이 `innerTimerRef`를 clear
- `setOpacity(1)` 타이머가 취소되어 opacity가 0에 머무름

**수정:**
- 파일: `components/create-flow/CreateWizard.tsx`
- 방법: fade-in을 별도의 `useEffect`로 분리하고 `requestAnimationFrame` 사용
- 상태: **수정 완료 + 검증 통과**
